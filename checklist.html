✓Конфиг деплоя render.yaml в корне репозитория
·Деплой на Render.com — выполнить вручную (см. примечание)
·Прописать секреты в GitHub: LLM_API_KEY, MCP_API_KEY · переменные: MCP_SERVER_URL, LLM_MODEL, LLM_BASE_URL

Деплой на Render.com: Settings → New Web Service → Connect repo → Render найдёт render.yaml автоматически. В дашборде Render: Environment → добавить MCP_API_KEY. После деплоя: скопировать URL сервиса в GitHub → Settings → Variables → MCP_SERVER_URL. Без MCP: агент работает через локальный fallback (файлы style_guide/).
