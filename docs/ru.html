Doc Reviewer: анализ качества документации с использованием LLM

Doc Reviewer автоматически сканирует вашу техническую документацию — PDF, Word‑документы, файлы Markdown и веб‑страницы — обнаруживает разделы с инструкциями и оценивает каждый из них по настраиваемому набору критериев качества с помощью больших языковых моделей.
Для каждой инструкции программа выводит оценку и рекомендации о том, что можно улучшить.

⬇️Установка

Скачайте и настройте Doc Reviewer на Windows, подключите LLM и запустите первую оценку.

⚡Быстрый старт

Загрузите первый документ и получите результаты оценки за несколько минут.

📄Основные концепции

Изучите проекты, документы, результаты оценки и наборы критериев.

⚙️Настройка

Подключите OpenAI, Anthropic, Ollama или любую совместимую с OpenAI модель.

Как работает​

1
Установка и запуск

Скачайте doc-reviewer.exe и запустите его. Программа запускает локальный сервер и автоматически открывает его в браузере.

2
Выбор и настройка модели LLM

В Настройки → Модели выберите модель из списка или добавьте свою. Если модель требует ключ, нажмите Edit и введите API‑ключ. Вы также можете настроить локальную модель через Ollama.

3
Создание проекта и загрузка документов

Создайте проект для продукта, затем загрузите файлы PDF, DOCX, Markdown или TXT — либо загрузите страницы по URL.

4
Запуск оценки и просмотр результатов

Нажмите Оценить, чтобы проанализировать все разделы с инструкциями. Просмотрите цветовые оценки и рекомендации, затем экспортируйте их в XLS.

Ключевые возможности​

📄Поддержка множества форматов

Парсите и оценивайте PDF, DOCX, Markdown и TXT файлы. Загружайте веб‑страницы по URL с помощью headless‑браузера.

🔍Автоматическое обнаружение инструкций

Doc Reviewer автоматически идентифицирует разделы с инструкциями в ваших документах с помощью морфологического анализа — ручная разметка не требуется.

✅Настраиваемые критерии оценки

Используйте встроенный набор критериев или создайте свой. Редактируйте критерии прямо в интерфейсе в формате Markdown.

🔀Сравнение версий через снимки

Сохраняйте результаты оценки в виде снимков и сравнивайте их между версиями документа, чтобы отслеживать улучшения.

🧠Контекст продукта для точной оценки

Добавьте описание продукта к проекту — LLM учтёт терминологию и аудиторию продукта при оценке инструкций.

🔒Локальная работа и конфиденциальность

Все данные хранятся на вашем компьютере. Doc Reviewer использует локальную базу данных SQLite и работает полностью офлайн, за исключением вызовов LLM API.

📄Автоматический глоссарий продукта (YAKE)

При загрузке документа Doc Reviewer извлекает глоссарий ключевых терминов продукта с помощью YAKE и включает его в подсказки LLM, уменьшая количество ложных предупреждений.

🔄Умная переоценка при обновлении документа

При замене документа неизменённые разделы копируются из предыдущих оценок, частично изменённые разделы получают подсказки о различиях, а только новый контент переоценивается, экономя вызовы LLM.
