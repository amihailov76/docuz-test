## Установка приложения

Вам предлагается скачать актуальную версию приложения со страницы загрузок.

Вы должны выполнить настройку системы перед первым запуском. Необходимо отметить,
что приложение поддерживает различные форматы входных данных.

Требуется установить зависимости. Для корректной работы необходимо,
чтобы на компьютере была установлена Java версии 11 и выше.

## Работа с проектами

Рассмотрим основные операции с проектами. Очевидно, что для создания
проекта вам следует нажать кнопку **Создать проект**.

В большинстве случаев достаточно стандартных настроек. Практически
все параметры можно изменить позже.

Запрещается использовать специальные символы в названии проекта.
Не допускается создавать два проекта с одинаковым именем.

## Экспорт данных

Для корректной работы экспорта система осуществляет проверку формата файла.
Приложение производит анализ структуры документа в процессе работы.

Смотрите раздел [Экспорт](/ru/workflows/export) для получения подробной информации.

Вам предлагается использовать гибкий и удобный интерфейс для настройки параметров.
Это мощный и надёжный инструмент для работы с документацией.

## Устранение неполадок

В некоторых случаях может возникнуть ошибка подключения. Фактически,
это происходит при отсутствии сети. Система думает, что соединение установлено,
но в любой момент времени может потерять его.

Вам следует обратиться в службу поддержки, если проблема сохраняется.
